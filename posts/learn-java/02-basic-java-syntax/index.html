Java學習筆記02-Java基本語法

2020-07-11
Learn-Java
Comments

一開始學習語言，當然要先認識Java基本語法。

推薦使用一些免費的Java基本教學課程，無論是程式語言新手還是已經其他程式語言經驗的人都很適合。

  • codecademy的Learn Java：有網頁版的編輯器與Terminal，可以邊學邊實作
  • educative的Learn Java from Scratch：感覺講得比codecademy的Learn Java再詳細一點
  • paizaラーニング：Java入門編：日文，比較活潑一點的學習方式

就看課程內容跟學習方式哪個中意就選哪個。

我選了codecademy，每天一小時左右，大概花了一禮拜學習完基本的課程。
