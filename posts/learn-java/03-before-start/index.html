IntelliJ IDEA

官方下載：https://www.jetbrains.com/idea/

IntelliJ IDEA有分Ultimate與Community兩個版本，我使用的是Ultimate版本。

SDK設定

安裝完後，需要讓IntelliJ IDEA知道要用哪個Java版本。

初次啟動，來到下面的歡迎畫面後，Configure→Structure for New Projects。

點下去後會開啟下圖的視窗，點選左側Platform Settings的SDKs，然後按一下上面的＋→Add JDK...。

按下後跳出視窗，要選擇剛剛裝好的Java資料夾，比較快的方式是按下command+shjift+g後直接輸入位置~/.sdkman/candidates/java/current，會自動跳到預設的Java資料夾，確定沒問題後點選Open。

成功讀取後會像下圖這樣子。沒問題就按OK。

設定到這基本的就完成了，之後再補一些方便開發的設定。

Spring initializr

方法ㄧ：從網頁上選擇後下載並用IDE打開

在Spring initializr上點選想要的Spring boot版本與框架的各種設定。

Project：分為Maven和Gradle，Maven使用XML來設定，Gradle就比較直觀一點。不知道選什麼的話就先選Gradle吧！

Language：因為要用Java開發，所以選Java

Spring Boot：有括弧的(像是SNAPSHOT、M1等)是不安定版。一般情況選擇沒有括弧的安定版。

Project Metadata：

  • Group：假設你要開發一個網站abc.com，這裡就填com.abc
  • Artifact：在abc.com下本次要開發的工具或app，就填在這（例如要開發登入的套件就可以填login）
  • Name：自動填入（Artifact）
  • Description：任意
  • Package name：自動填入（{Group}.{Artifact}），同時是到時候開發時的package路徑
  • Packaging：Jar
  • Java：選擇安裝的Java版本

Dependencies：已經知道的依賴package可以從這裡選，不選的話也可以之後自行加上去

自行加上去spring.datasource.url後面追加?useUnicode=yes&characterEncoding=UTF-8防止寫入資料庫時亂碼）

通常會裝：

Spring Web：Controller GET method Web必裝

Spring Boot DevTools：自動重啟，IDE開發加速用

MySQL Driver：連Mysql用的驅動，要連其他類型的資料庫可以打關鍵字找

Spring Data JPA：Entity、Reposertoy、Service的套件

另外：

Thymeleaf：要回傳HTML內容的時候的template套件，也有其他可以選擇不過目前只用過這個

好了之後就按GENERATE，就會把zip檔案下載下來。解壓縮後直接用IntelliJ IDEA打開，IDE裝完環境後就可以開始開發了！

2020/08/30追記：Project作成後，在build.gradle可以再追加Dependencies。追加後需要Reload（./gradlew --refresh-dependencies 或 IntelliJ IDEA 側邊欄的Gradle -> Reload All Gradle Projects）

方法二：直接用IntelliJ IDEA作成Project

另外在IntelliJ IDEA裡也可以直接使用Spring initializr來開啟新專案，填的內容就跟網頁上一樣。

點選Create New Project→左側的Spring initializr→確認JavaSDK後Next
