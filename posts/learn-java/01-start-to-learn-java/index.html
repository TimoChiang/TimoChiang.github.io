Java學習筆記01-開始學Java

2020-07-11
Learn-Java
Comments

之前做了什麼：

  • 出社會後當了幾年的PHP web developer
  • 工作上偶然碰到Golang，因為有興趣而自學了幾個月，若有時間跟機會還會想繼續精進

為何要學Java

  • 工作好找
  • 大規模的服務滿多用到Java的
  • 因為歷史久，語言相對成熟，且網路資源也多
  • 有不少預設的資料結構可以使用，對於非本科的我來說可以對這一部分更熟悉

半年學習目標

  • Java語法與資料結構熟練→用Java刷題可以得心應手
  • Spring Boot 框架熟練→做出Web、RESTAPI的作品
  • 想到再加上
